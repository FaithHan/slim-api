# slim-api

SlimFramework整合NotORM、MongoDB、Predis，用于API开发，支持CLI-Command

### 使用：

```sh
#获取框架和所依赖的库
composer update
```

```sh
#cli-command
php cli greet IIInsomnia

#output
Hello IIInsomnia
```

* Point your virtual host document root to your new application's `public/` directory.
* Ensure `logs/` is web writeable.
